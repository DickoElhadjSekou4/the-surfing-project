# The Surfing Project

Projet réalisé dans le cadre de la formation ETP4.

## Description

The Surfing Project est une landing page responsive développée avec Bootstrap 5.

L’objectif du projet était de reproduire une page web pour une école de surf décentralisée basée sur le peer-learning, en respectant les contraintes suivantes :

- Utilisation de Bootstrap
- Utilisation minimale de CSS personnalisé
- Approche mobile-first
- Mise en page responsive

## Technologies utilisées

- HTML5
- Bootstrap 5 (via CDN)
- Font Awesome
- GitHub Pages pour le déploiement

## Structure du projet

La page contient :

- Une navbar responsive
- Une section Hero avec image de fond
- Une section de présentation avec icônes
- Une section témoignages utilisant des cartes Bootstrap
- Une section offres/prix avec cartes et boutons
- Un footer

## Objectifs pédagogiques

Ce projet permet de :

- Comprendre la structure d’un document HTML
- Maîtriser la grille Bootstrap (container, row, col)
- Utiliser les composants Bootstrap (navbar, card, button)
- Mettre en place un site responsive
- Déployer un site statique avec GitHub Pages
